# Multilingual Live Translation Keyboard

## Overview

This repository contains the Product Requirements Document (PRD) and prototype for a **Multilingual Live-Translation Keyboard**, a mobile keyboard solution designed to enable seamless communication across different languages in social messaging applications.

The concept allows users to type in their native language while messages are automatically translated into the recipient's language in real time.

---

## Problem Statement

Global communication is increasingly happening through messaging apps, yet language barriers still prevent seamless interaction between users who speak different languages.

Most existing solutions require switching between translation apps or manually copying text, which disrupts the conversation flow.

---

## Proposed Solution

The **Multilingual Live-Translation Keyboard (MLTK)** integrates translation directly into the mobile keyboard.

Key capabilities include:

* Real-time bidirectional translation
* Automatic language detection
* Cross-app keyboard integration
* Language switching within conversations
* Message preview before sending

The keyboard works across messaging platforms such as:

* WhatsApp
* Instagram
* Facebook Messenger
* Telegram
* Signal

---

## Key Features

* Real-time text translation
* Language selection and switching
* Translation preview before sending
* Option to view original message
* Standard keyboard experience with predictive typing
* Privacy-focused architecture
